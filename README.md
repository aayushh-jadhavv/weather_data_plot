# Weather data plotted for 4 cities

To see for yourself, clone the repo: 

`git clone ...`

Or just download and unzip the zip file.

And then install pandas, numpy and matplotlib on your system.

*for city-based plots of temperature*

`python city_spec.py *city_name`

where city_name is optional and prints csv for chosen city (`sd` for San Diego, `ny` for New York,  `sj` for San Jose, `pa` for Philadelphia)

*for scatter plot of temperature, humidity, precipitation and wind speed across cities, at different times in the 2024 year*

`python param_grid.py`

Weather data CSV file from [Kaggle](https://www.kaggle.com/datasets/prasad22/weather-data?resource=download)
